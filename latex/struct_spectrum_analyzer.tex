A JUCE component that displays the audio spectrum analysis. The \mbox{\hyperlink{struct_spectrum_analyzer}{Spectrum\+Analyzer}} class is a component that listens to audio processing parameters and displays the audio spectrum analysis. It uses a \mbox{\hyperlink{struct_path_producer}{Path\+Producer}} to generate paths from the audio data, and draws these paths on the component.
