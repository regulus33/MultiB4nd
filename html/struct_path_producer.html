Detailed Description

A class that processes FFT data and generates paths. PathProducer class processes FFT data and generates paths that can be used to render audio data. It uses a LeftChannelFifo buffer to store incoming audio data, a monoBuffer to store the mono representation of the audio data, and LeftChannelFFTDataGenerator to generate FFT data.
