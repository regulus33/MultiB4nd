 Resizes the GlobalControls component This method is responsible for laying out the child components of the GlobalControls component. The child components are sliders for gain input, low-mid crossover frequency, mid-high crossover frequency, and gain output. A FlexBox layout is used to arrange the sliders in a row. GlobalControls::GlobalControls ( juce::AudioProcessorValueTreeState &  apvts)

The constructor for the GlobalControls class. This constructor creates 4 sliders (input gain, low-mid crossover frequency, mid-high crossover frequency, output gain) and attaches them to the corresponding parameters in the AudioProcessorValueTreeState object. The sliders are also given labels and label values based on the AudioParameter objects they are attached to. The sliders are added to the component and made visible.

Parameters
apvtsAn AudioProcessorValueTreeState object that holds the parameters used by the sliders.
