Detailed Description

template<typename BlockType>
struct FFTDataGenerator< BlockType >

Lets look at a simple JS audio visualizer to understand FFT visualization There will be some FFTGenerator class that produces an array. Lets call it frequencyData.

const frequencyData = new Uint8Array(analyserNode.frequencyBinCount);
analyserNode.getByteFrequencyData(frequencyData);

Each value in the frequencyData array represents the amplitude of a particular frequency band in the audio signal at brief moment in time. The AnalyserNode.frequencyBinCount property specifies the number of frequency bands used for analysis, and the frequencyData array has a length equal to AnalyserNode.frequencyBinCount.

Frame 1 `frequencyData = [ 23, 45, 68, 90, 112, 134, 155, 177, 198, 220, 241, 255, 250, 236, 223, 210, 197, 184]`
Frame 2 `frequencyData = [ 33, 55, 78, 100, 122, 144, 165, 187, 208, 230, 251, 265, 260, 246, 233, 220, 207, 194]`
Frame 3 `frequencyData = [ 53, 75, 98, 120, 142, 164, 185, 207, 228, 250, 271, 285, 280, 266, 253, 240, 227, 214]`
Frame 4 `frequencyData = [ 83, 105, 128, 150, 172, 194, 215, 237, 258, 280, 301, 315, 310, 296, 283, 270, 257, 244]`
Frame 5 `frequencyData = [ 123, 145, 168, 190, 212, 234, 255, 277, 298, 320, 341, 355, 350, 336, 323, 310, 297, 284]`

The array is the size of exactly the amount of BANDS that FFT determines are existing in the current signal source. Then frame by frame we iterate through our array

for(let band = 0; band < frequencyData.length; band++)
{
drawBarWithHeight(band);
}

Now look at graph below. X axis is index of array (band) and Y axis is the amplitude of that band.
