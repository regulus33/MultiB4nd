Detailed Description

A JUCE component that displays the audio spectrum analysis. The SpectrumAnalyzer class is a component that listens to audio processing parameters and displays the audio spectrum analysis. It uses a PathProducer to generate paths from the audio data, and draws these paths on the component. The component also handles redrawing the background grid and text labels as needed.
